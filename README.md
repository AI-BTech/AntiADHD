# AntiADHD
a couple of lil scripts to let you focus. i made this because i can never focus myself because of good old ADHD....
# How to install
there are 2 ways to install, the advanced way, and the easy way.
> this assumes that if you know about this, your technical enough to have Python, PIP, and Git on there.
### Dependencies
1. Python 3.8+ (tested on the latest)
2. Git
3. a terminal (why wouldnt you?)
4. Make  (for wev)
### how to install (easy way)
1. run `git clone https://github.com/AI-BTech/AntiADHD`
2. run `cd AntiADHD`
3. run `python3 antiadhdinstaller.py`
4. go through the installer
5. **done!**
